DigiAgenda - UI/UX

Este documento visa saber quem é nosso cliente, conhecer o público-alvo, nosso usuário ideal através do Persona e do Mapa de Empatia além de, apresentar a interface do usuário atráves dos mockups.

1. Perfil do usuário

Persona é um perfil fictício construído para representar o cliente ideal de uma empresa, criado para ajudar a compreendermos melhor quem é o cliente e o que ele precisa. Já o a mapa de empatia é uma ferramenta onde monta-se os perfis de segmentos de clientes atendidos, de forma possa ser compreendido seu ambiente, seus comportamentos, preocupações e aspirações.
